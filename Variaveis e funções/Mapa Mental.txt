                                        Tipos de Dados de Referência em JavaScript
Objetos
Objetos em JavaScript são coleções de propriedades
Cada propriedade é uma associação entre um nome e um valor
Analogia: uma mochila cheia de itens, onde cada item tem um nome e um valor
Acessar propriedades de um objeto:
Usando a notação de ponto (objeto.propriedade)
Usando a notação de colchetes (objeto['propriedade'])
Deletar propriedades de um objeto:
Usando o operador delete (delete objeto.propriedade)
Funções
Funções são blocos de código projetados para executar tarefas específicas
Analogia: uma máquina de suco, que recebe uma entrada (fruta), processa e retorna uma saída (suco)
Sintaxe básica de uma função:
function nomeDaFuncao(parametros) { código a ser executado }
Funções servem para facilitar a execução de tarefas e podem ser reutilizadas
Exemplo: Função para calcular o IMC
Recebe um objeto Pessoa com altura e peso
Calcula o IMC e retorna um objeto com o resultado
Pode ser reutilizada para diferentes pessoas
Aplicação Prática
Criação de uma função calculateIMC para calcular o IMC de uma pessoa
A função recebe um objeto Pessoa com nome, peso e altura
Retorna um objeto com o IMC, o resultado (saudável ou não) e uma mensagem
A função pode ser reutilizada para calcular o IMC de diferentes pessoas
